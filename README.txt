Welcome to Danesh Theme 2

This is for educational purposes only.  

This is full ecommerce theme designed for word press and woo commerce.

It is fully responsive. The menu can have either a drop down menu or mega menu. 
To use mega-menu add Class mega-menu on first level menu link
To use drop down menu add class drop-menu on first level menu link.

To add to banner, create a post and add featured tag.

To ensure functionality of Special, Popular, New, Clearance thumbs on home page, create
respective categories.

This theme needs installation of woo-commerce plug in and jet pack.
